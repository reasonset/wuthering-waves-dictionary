# Wuthering Waves Disctrionary

## Synopsis

Kuro GamesのオープンワールドアクションRPGゲーム「鳴潮」で登場する用語のMozc/Google日本語入力向け辞書

## Description

Mozc (Google日本語入力) 形式の辞書ではあるが、ユーザー辞書はおおよそ決まったフォーマットであれば識別できるようになっていることが多いため、おそらく他のソフトウェアでも利用可能である。

一般的な語でもMozcだと変換できない場合があるため、一部の語は鳴潮用語でなくても収録している。

## Usage

`ww.txt` または `ww-all.txt` を辞書ツールでインポートする。
また、必要に応じて `wwcommon.txt` もインポートする。

`src/*` を個別にインポートすることも恐らく可能だが、検証はしていない。

## 収録基準

* 変換が難しい、よく使うけどすんなり変換できないものを中心に収録
* プレイブルキャラクターはよくあるカタカナ名であっても収録
* 一部正式名称が非常に長いもの(例: 死の歌が纏う海の廃墟)の一部は短い名称(例: はいきょ)でも変換できるように登録
* 「ゔ」が入力できない環境向けに「ヴ」をふくむものはば行でも変換できるように登録 (例: べりーな → ヴェリーナ)

なお、カタカナ名を登録しないのは比較的簡単に変換が可能で、なおかつ登録すると対象が増えて大変なので登録していないだけで、contributeで追加される分には受け入れる。

読みの揺れについては、公式に読みが明らかになっているものは基本的には収録しないが、若干の揺れを吸収できるように収録する場合もある。
例えば、「ウインド/ウィンド」(公式にも表記ゆれが存在する)の読みの「ういんど/うぃんど」などである。
公式読みがない場合は、現実的に考えられる範囲で表記ゆれを収録する場合がある。

## 登録順

1. 雑多な用語
2. ストーリーにのみ登場する語
3. 地名 瑝瓏/ブラックショア/リナシータ/その他, 組織名
3. 周期コンテンツ
    1. 恒常周期コンテンツ / 短縮形
    2. 不定期開催イベント
5. 戦闘用語
6. システム用語
7. バッグ
    1. システム系
    2. デバイス関連
    3. 突破素材
    4. 花
8. 武器 - 長刃/迅刀/拳銃/手甲/増幅器
9. 残像
    1. 津波級
    2. 怒涛級
    3. 巨浪級
10. 音骸 / 短縮形
11. プレイアブルキャラクター
12. NPC
13. その他人名
14. キャラクターの説明に出現する語

「NPC」「キャラクターの説明に出現する語」は使う機会があったものから順に少しずつ追加している。

## Gitタグ

Gitのタグのバージョンは、本ソフトウェアのバージョンではなく、対応する鳴潮のバージョンである。

恐らく使う機会はほとんどない。

## 統合辞書の種別

src/*`以下がソース辞書である。
現状ではそれぞれが独立して辞書として扱えるものではあるが、これは*将来に渡っては保証されない。*
主に、管理が難しくなってきたらコメント機能を追加することが考えられており、この場合`src/*`をインポートすることはできなくなる。

このため、基本的にはルートに置かれている辞書ファイルをインポートすることが想定されているが、現状は以下の種類がある。

|ファイル|内容|
|------|------------------------|
|`ww-all.txt`|現在登録されているすべての語(`src/`以下)を統合したもの|
|`ww.txt`|キャラクターの「レシート」内の語を除外したもの|

「レシート」要素は予測変換の邪魔になる場合があるため、分けられている。

`commonname/*`は正式な語ではなく、プレイヤー側の俗語で、かつ一般に定着している汎用的な語を収録したものである。
例えば「いし→金髄の波模様」は収録するが、「くさ→ヴェリーナ」は収録しない。

これは通常の鳴潮用語とは全く別種のものであるため、別途`wwcommon.txt`として統合されている。
